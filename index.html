How To Use This Predictor

Where To Get Match Data

  • Flashscore
  • SofaScore
  • WhoScored
  • Transfermarkt
  • FBref

What Each Input Means

  • H2H Wins = Head to Head wins in last 5 meetings
  • Recent Wins = Wins in last 5 matches
  • Avg Scored = Average goals scored in last 3 games
  • Avg Conceded = Average goals conceded in last 3 games
  • Key Player = Important player available
  • Coach System = Team tactical formation
  • Trophies = Total major titles team has won

Example Using AI

Ask AI: 
Give statistics for Arsenal vs Chelsea:

1. Last 5 head to head wins
2. Last 5 match results
3. Average goals scored last 3 matches
4. Average goals conceded last 3 matches
5. Key player availability
